## Rodando a Aplicação

Certifique-se que o build system `meson` e a biblioteca `gmp` estejam instalados no seu sistema.
Em seguida:

```bash
meson build
meson -C build
./build/rsa
```

A aplicação poderá então ser acessada por `https://localhost:3000`.
